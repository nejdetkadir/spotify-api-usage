# Spotify API Usage
An app that showcases how to use Spotify API with VueJS

![cover](doc/cover.png)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
